# Time Series Forecasting: US Renewable Energy Production

This project focuses on developing a predictive model for total renewable energy production in the United States, based on data provided by the \textb{U.S. Energy Information Administration (EIA)}.

## Objective
The goal is to identify, estimate, and validate an optimal ARIMA model to generate reliable forecasts. These insights are designed to support strategic decision-making in infrastructure planning and resource management.

## Key Features
Time Series Analysis: Comprehensive study of renewable energy production trends.

Modeling: Identification and validation of ARIMA models.

Forecasting: Future production estimation to support supply management.

## Technologies Used
R (Statistical modeling and data processing)

LaTeX (Technical reporting)

## Files in this Repository
Time_Series_Analysis.R: Full R script containing the statistical modeling.

Final_Report.pdf: Detailed technical report explaining the methodology and results.
